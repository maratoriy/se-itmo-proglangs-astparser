# se-itmo-proglangs-astparser
C parser of infix notation expression to AST tree, its evaluation, RPN printer

In current version:
  - logical operators:
    - '&&', '||', '~', '->, '<->'
  - math operators:
    - '*', '/', '+', '-', '%'
  - unary math operators:
    - '-', '!'(factorial)
  - brackets '(' and ')'
